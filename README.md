1. 프로젝트 목적: RayTracing 렌더링 기법을 학습하고 이를 적용하여 간단한 렌더링 구현


2. 공부 기간: 2024/04/24 ~ 2024/04/25 (약 2일)


3. 사용한 기술: C++, 비주얼스튜디오


4. 주요 기능:


    Sphere를 RayTracing 렌더링


    Lighting을 적용해 조명 효과 표현
  
  
    Shadow를 적용해 그림자 표현
  
  
    Reflection을 적용해 물체 간 반사를 표현
  
  
    Refraction을 적용해 굴절 효과 표현
